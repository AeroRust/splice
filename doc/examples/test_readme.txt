Examples of SPLICE assembly programs from OPS-SAT test suit:
test_a1.splc: basic arithmetics: multiply-add operation
test_a2.splc: sin/cos and divide/subtract
test_a3.splc: tan and divide/subtract
test_a4.splc: power and logarithms
test_a5.splc: bitwise NOT-OR operation and binary output format

test_b1.splc: internal timers and ADCS mode display
test_b2.splc: show ADCS vectors (magnetometer, quaternions, sun vector)
test_b3.splc: switch into Nadir-pointing ADCS mode, for a given GPS coordinates
test_b4.splc: make a photo of given target, if the ADCS mode is nadir-pointing
test_b5.splc: calculate an absolute magnitude of the magnetic field

test_c1.splc: get a vector of altitude and dAlt/dT values
test_c2.splc: 2x2 matrix multiplication
test_c3.splc: cross-task data loading (load and transpose matrix)
test_c4.splc: orbital period calculation
test_c5.splc: linear filter (for spacecraft altitude value)

test_d1.splc: max value
test_d2.splc: min value
test_d3.splc: eccentricity calculation
test_d4.splc: FPU constants test
test_d5.splc: Log debug level switching test
